HTML <embed> Tag

The <embed> tag is used to **embed external content** such as videos, audio, PDFs, or other media directly into a webpage.

üìå Common Uses:

  • To display PDF documents
  • To embed audio or video players
  • To include multimedia plugins or Flash content (deprecated)

üí° Syntax:

<embed src="file-path" type="media/type" width="width" height="height">

üéØ Attributes:

  • src: Path to the embedded file (e.g., "video.mp4" or "sample.pdf")
  • type: MIME type of the content (e.g., "application/pdf", "video/mp4")
  • width & height: Dimensions of the embedded area

‚úÖ Example: Embed a PDF

<embed src="sample.pdf" type="application/pdf" width="100%" height="400px">
üîî Note: The <embed> tag is an empty tag and does not require a closing tag. It is not supported in some older browsers.

üé® Styling with CSS

You can wrap the <embed> element inside a div and style the container or control its size using CSS.
