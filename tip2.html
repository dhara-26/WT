HTML <body> Tag

The <body> tag defines the **main content** area of an HTML document that is visible to users in a web browser. It holds all elements like headings, paragraphs, images, links, tables, lists, etc.

üìå Key Points:

  • Only one <body> tag is allowed per page.
  • All visible content must go inside this tag.
  • It appears after the <head> tag and before </html>.

üí° Basic Syntax:

<html>
  <head>
    <title>My Web Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

You can apply CSS styles directly to the <body> tag to set background color, fonts, margins, padding, and more.

‚úÖ Example:

  background-color: #f0f0f0;
  color: #333;
  font-family: Arial, sans-serif;
  padding: 20px;
}
This content is inside the <body> and is styled using CSS.
