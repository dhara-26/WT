ANCHOR TAG(<a>...</a>)

The anchor tag (<a>) in HTML is used to create hyperlinks. It allows users to move from one page to another or to a specific section within a page. The href attribute in the anchor tag defines the link's destination.

Uses of the anchor tag:
- To link one webpage to another (like Home to About page)
- To open external websites (e.g., Google)
- To link to downloadable files like PDFs or documents
- To jump to a specific section within the same page using IDs
- To open links in a new tab using target="_blank"
