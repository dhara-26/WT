What is a Form?
A form in HTML is used to collect user input and send it to a web server for processing. It acts as an interface between the user and the application, allowing people to enter data like names, email addresses, passwords, and more.

Uses of Forms:
  • To register users on websites
  • To log in or sign up for services
  • To collect feedback or contact information
  • To place orders or fill surveys
  • To search within a website or database
